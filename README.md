# simple_parser
test_assignment

1) Клонируем репозиторий:
```
git clone https://github.com/RussianProgram/simple_parser
```

2) Настраеваем окружение 
```
cd simple_parser
poetry shell
poetry install
```

3) Запускаем парсер
```
cd src
python main.py
```

## В файле config.py настраеваем нужное кол-во страниц для парсинга.
